## 二、项目简介（200字内）

AI 商务谈判官是一个基于小米 MiMo 多模态大模型的智能商务谈判分析系统。系统采用 **8 个 AI Agent 协作** 的架构，用户上传合同/邮件/招标文件后，系统自动完成文档分析、风险评估、策略制定、对手模拟、法律审查、情绪分析、市场情报收集，最终生成综合谈判报告和 2 分钟语音简报。

核心亮点：
- **8 Agent 流水线协作** — 业界领先的多 Agent 编排架构
- **同时调用 MiMo 三模型** — Omni（视觉）+ Pro（推理）+ TTS（语音）
- **覆盖 10 个行业** — 内置行业基准数据和风险模式库
- **零依赖部署** — 仅使用 Node.js 内置模块，开箱即用

## 六、项目亮点与创新点

### 6.1 技术创新
1. **8 Agent 流水线编排** — 业界少见的多 Agent 协作架构，支持并行执行和可选步骤
2. **三模型协同** — 同一任务链中无缝调用 Omni（视觉）+ Pro（推理）+ TTS（语音）
3. **零依赖部署** — 仅使用 Node.js 内置模块，无需 npm install，开箱即用
4. **自研中间件** — 完整的认证/限流/日志系统，无需第三方库

### 6.2 业务创新
1. **中国法律深度集成** — 内置《民法典》《合同法》《个人信息保护法》等法条知识库
2. **心理博弈识别** — 基于 PAD 情绪模型和行为经济学理论的谈判心理分析
3. **行业数据驱动** — 覆盖 10 个行业的基准价格、风险模式、法律合规要求
4. **条款模板库** — 6 类标准合同条款模板，支持快速审查和修改建议

### 6.3 工程创新
1. **JSON 文件数据库** — 零编译、零配置的数据持久化方案
2. **多格式导出** — 支持 Markdown、HTML、JSON、CSV 四种导出格式
3. **Demo 模式** — 无需 API Key 即可完整演示所有功能
4. **完整测试覆盖** — 685 行单元测试，覆盖所有核心模块
